What is a Sigma Rule?

      

        Sigma is a generic, open signature format for SIEM systems — like Snort/Yara for log files. A single Sigma rule can be converted to SPL (Splunk), KQL (Elastic), AQL (QRadar), or DNIF query syntax using the sigma-cli converter tool. This lets you write detection logic once and deploy across any SIEM platform.
        


        Convert to Splunk SPL: sigma convert -t splunk rule.yml

        Convert to Elastic KQL: sigma convert -t es-qs rule.yml

        Convert to QRadar AQL: sigma convert -t qradar rule.yml
